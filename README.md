### Prerequisites

- jQuery >= 2.x
- iziModal >= 1.6.0
- deepmerge >= 4.x

### Prerequisites (Development on iziModalWrap)

- Typescript (tested with 3.7 branch, more to come)

### Install

- `npm i --save-dev izimodal-wrap` or `npm i --save izimodal-wrap`
